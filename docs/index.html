Falcon Framework Docs

Welcome to the official Falcon Framework documentation. Here you will find information on configuring falcon framework, the directory structure and which files are responsible for what. The documentation also includes information about the paging and login system, and details how the controller interacts with the view as well as the model.

Code also contains documentation in the form of comments for clarification in the framework's files. Each file also has it's own heading comment that details what the file is and what should and shouldn't be edited. If you wish to edit falcon framework's core code it's recommended that you read these comments. If you wish to modify areas that the comments recommend not to modify, feel free to do so but the developers of Falcon Framework are not responsible for any downtime or issues caused by this.

If worst comes to worst and it seems you've broken the framework beyond repair, backup important files and reinstall a clean version from the GitHub repo.
